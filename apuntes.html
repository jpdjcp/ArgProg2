Formato de texto HTML

  • n - Esto es texto en negrita
  • i - Esto es texto en itálica/cursiva
  • u - Esto es texto subrayado
  • strike - Esto es texto tachado
  • tt - Esto es texto monoespaciado
  • mark - Esto es texto resaltado>
  • small - Esto es texto pequeño
  • del - Esto es texto borrado
  • ins - Esto es texto insertado
  • sub - Esto es texto con subíndice
  • sup - Esto es texto con superíndice
